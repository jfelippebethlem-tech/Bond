## 4. Tarefa nociva neutralizada: `BondLikersSemanal`

- **O quê:** tarefa agendada do Windows (sexta 09:00) que rodava `bond-likers.ps1 -Scheduled`
  → `scripts/captura-likers.mjs`, o método CAPADO (~100), com `IG_INTERACTIVE=false`,
  **sobrescrevendo o contrato de produção (`likers-sync`) toda sexta**.
- **Não foi possível desabilitar a tarefa** daqui: `Disable-ScheduledTask` exige admin ("Acesso negado").
- **Solução (sem admin):** guard no topo de `bond-likers.ps1` — quando chamado com `-Scheduled`,
  imprime aviso e `exit 0` **sem capturar e sem tocar no contrato**. Testado: exit 0, no-op.
  Execução manual (sem `-Scheduled`) continua funcionando, caso alguém queira o método antigo.
- **Para REMOVER a tarefa de vez (quando tiver admin):**
  ```powershell
  Unregister-ScheduledTask -TaskName "BondLikersSemanal" -Confirm:$false
  ```
  Depois disso, o guard em `bond-likers.ps1` pode ser removido se quiser.
